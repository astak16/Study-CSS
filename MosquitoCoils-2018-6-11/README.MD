### 代码理解
1. 定位：父元素`position：revative;` 子元素`position:absolute;`
2. 完美居中
```
  display: flex;
  align-items: center;
  justify-content: center;
```
3. 渐变
  ```background: radial-gradient(circle at center,midnightblue, black);```
4. 半圆`border-radius: 50% 50% 0 0 /100% 100% 0 0;`
